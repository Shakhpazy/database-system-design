# Database System Design

A simple database system design project.

## How to Run

Run `python main.py`

To exit the program, press Ctrl + Z.

## Supported Queries

The assignment requires these queries to work:

- CREATE DATABASE db_name;
- USE db_name;
- DROP DATABASE db_name;
- CREATE TABLE tb_name (col1 type, col2 type, ...);
- DROP TABLE tb_name;
- ALTER TABLE tb_name ADD col type;
- SELECT * FROM tb_name;
